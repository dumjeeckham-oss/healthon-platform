# 20. CTO Notes

Home 화면은 건강ON의 가장 중요한 화면이다.

사용자는 앱을 실행한 후 3초 안에

- 오늘 걸음 수
- 누적 거리
- 목표 달성률
- 내 순위

를 확인할 수 있어야 한다.

공지와 커뮤니티는 두 번째 페이지로 배치하여 첫 화면을 최대한 단순하게 유지한다.

향후 새로운 챌린지가 추가되어도 Home 구조는 변경하지 않고, 챌린지 카드만 추가하는 방식으로 확장한다.

모든 화면은 Flutter Feature 기반으로 개발한다.
