# 1. 프로젝트 개요

## 프로젝트명

건강ON (HealthON Platform)

## 슬로건

건강을 켜다, 함께 걷다.

## 프로젝트 목표

건강ON은 부천의료복지사회적협동조합이 운영하는 건강 참여 플랫폼입니다.

첫 번째 서비스는 **부천100K 걷기 챌린지**이며,
향후 금연, 출자금 증좌, 건강검진, 플로깅, 자전거 타기 등
다양한 참여형 챌린지를 하나의 플랫폼에서 운영할 수 있도록 설계합니다.

---

# 2. 프로젝트 비전

건강ON은 단순한 걷기 앱이 아닙니다.

건강ON은

- 조합원 건강관리
- 시민 참여
- 공동체 활성화
- 건강 데이터 기반 통계
- 관리자 업무 자동화

를 목표로 하는 플랫폼입니다.

---

# 3. 개발 철학

본 프로젝트는 아래 원칙을 따른다.

## Documentation First

모든 개발은 문서를 먼저 작성한 후 시작한다.

---

## AI Native Development

ChatGPT

Cursor

Codex

AI와 함께 개발하는 프로젝트를 목표로 한다.

---

## Data First

데이터는 프로젝트의 가장 중요한 자산이다.

코드는 다시 만들 수 있지만 데이터는 잃어버리면 안 된다.

---

## Mobile First

모든 기능은 모바일 사용자 경험을 우선으로 설계한다.

---

## Simple UX

60대 이상도 쉽게 사용할 수 있도록 설계한다.

# 10. CTO Notes

건강ON은 단순한 이벤트 앱이 아니다.

건강ON은 부천의료복지사회적협동조합의
장기적인 디지털 플랫폼 구축을 목표로 한다.

따라서

- 유지보수
- 확장성
- 데이터 보호
- 운영 편의성

을 항상 우선한다.

새로운 기능은 기존 구조를 변경하지 않고
추가할 수 있도록 설계한다.
