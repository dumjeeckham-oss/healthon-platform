# 📒 건강ON 의사결정 기록

이 문서는 프로젝트의 주요 의사결정과 그 이유를 기록합니다.

## 2026-07-01

### 프로젝트명

건강ON (HealthON)

**결정 이유**

- 건강과 참여를 쉽게 표현
- 향후 다양한 챌린지로 확장 가능

## 2026-07-01

### 플랫폼

Flutter

**결정 이유**

- Android / iOS 동시 개발
- 유지보수 비용 절감
- 건강 데이터 연동 지원

### Backend

Supabase

**결정 이유**
