# 📖 건강ON 운영 매뉴얼 (Operation Manual v1.0)

> Last Update : 2026-07-01

---

# 1. 목적

본 문서는 건강ON 플랫폼을 운영하는
부천의료복지사회적협동조합 담당자를 위한 운영 매뉴얼이다.

개발자가 없어도
챌린지를 개설하고,
참여자를 관리하며,
통계를 확인할 수 있도록 한다.

---

# 2. 운영자 권한

운영자는 다음 기능을 사용할 수 있다.

- 챌린지 생성
- 챌린지 수정
- 챌린지 종료
- 공지사항 등록
- 응원메시지 등록
- 참여자 조회
- 통계 확인
- 배지 관리
- 시즌 관리
- 엑셀 다운로드
- PDF 완주증 발급
- 관리자 권한 관리

# 부록

운영자가 가장 먼저 확인해야 하는 것은

"오늘 얼마나 많은 사람이 건강한 하루를 시작했는가"

이다.

건강ON은

관리 시스템이 아니라

건강 문화를 만드는 플랫폼이다.
