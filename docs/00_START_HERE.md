# 🚀 건강ON 프로젝트 시작하기

환영합니다.

이 저장소는 **건강ON(HealthON Platform)** 프로젝트의 공식 저장소입니다.

건강ON은 단순한 걷기 앱이 아니라, 부천의료복지사회적협동조합의 건강 챌린지 플랫폼입니다.

# 프로젝트 목표

- 건강증진
- 조합원 참여 확대
- 건강 데이터 기반 통계
- 관리자 업무 자동화
- 장기적으로 다양한 챌린지 운영
