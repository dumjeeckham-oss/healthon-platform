# healthon-platform
# 🟢 건강ON (HealthON Platform)

> **건강을 켜다, 함께 걷다.**

건강ON은 **부천의료복지사회적협동조합**이 운영하는 건강 참여 플랫폼입니다.

첫 번째 서비스는 **부천100K 걷기 챌린지**이며, 향후 금연, 출자금 증좌, 건강검진, 봉사활동 등 다양한 참여형 챌린지를 운영할 수 있도록 설계된 확장형 플랫폼입니다.

---

# 프로젝트 목표

* 걷기 챌린지 운영 자동화
* 건강 데이터 자동 수집
* 조합원과 시민이 함께하는 건강 플랫폼 구축
* 관리자의 운영 업무 최소화
* 장기적으로 조합의 디지털 플랫폼 구축

---

# 장기 목표

건강ON은 단순한 걷기 앱이 아닙니다.

건강ON은 지역사회 건강과 공동체 참여를 연결하는 디지털 플랫폼을 목표로 합니다.

향후 다음과 같은 챌린지를 지원합니다.

* 부천100K
* 금연ON
* 출자ON
* 건강검진
* 플로깅
* 자전거
* 봉사활동
